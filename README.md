# Follow-Up Desk

Follow-Up Desk is a simple React application for tracking job application follow-ups.
It allows users to add notes about companies, roles, and next steps, with data persisted in localStorage.

## Features

- Add new application follow-up notes
- Basic form validation
- Display a list of saved notes
- Data persistence using localStorage
- Clean and responsive layout

## Built With

- React
- JavaScript (ES6)
- CSS
- Vite

## What This Project Demonstrates

- Working with React hooks (`useState`, `useEffect`)
- Controlled form inputs
- Basic client-side validation
- Rendering lists dynamically in React
- Persisting data in the browser using localStorage
- Organizing a small React project cleanly
